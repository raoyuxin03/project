# Product

## Register

product

## Users

个人创作者使用 Agnes AI 在本地完成聊天、图片生成、视频生成、素材管理和画布编排。用户不希望学习底层节点或复杂模型参数，只需要清楚地输入提示词、上传参考素材、保存结果并继续创作。

## Product Purpose

Agnes Web 是一个本地 AI 创作工作台，把文本、图片、视频和资产流转封装在同一个界面里。成功标准是：用户能在“创作”页完成生成，在“资产”页复用结果，在“设置”页配置 API，而不需要理解后端接口细节。

## Brand Personality

清晰、安静、可靠。界面应该像一个稳定的创作控制台，优先让任务可完成、状态可理解、错误可诊断。

## Anti-references

不要做成营销落地页、过度装饰的模型展示页，或需要用户理解复杂 ComfyUI 节点的专业工具界面。避免把付费调用隐藏在不明确的按钮后面。

## Design Principles

- 生成流程优先：输入、引用素材、生成按钮、结果保存必须清楚。
- 设置即能力：模型、Base URL 和 API Key 要在设置页可见、可修改、可恢复默认。
- 付费调用可预期：涉及 Aixoras / Seedance / Image2 时，界面应明确模型来源和额度消耗风险。
- 结果自动入库：生成成功后默认进入资产和历史，减少手工搬运。
- 错误可修复：错误信息要指出缺 API Key、接口失败、权限不足或结果解析失败。

## Accessibility & Inclusion

默认按产品工具界面处理，保持足够文字对比度、键盘可操作的表单与按钮，并避免依赖动画表达关键状态。
